行测-数学计算-三大方法

三大方法：代入排除法、数字特征法、方程法

时间：20190830

参考：[粉笔教程]（https://www.bilibili.com/video/av64106883?p=8）

1. 代入排除法

（1）什么时候用

  • 题型

    年龄、余数、不定方程、多位数

  • 选项信息充分

    选项为一组数，选项可以转化为一组数。问法（分别/各）

  • 其他情况

    条件特复杂（题干长、主体多、关系乱），排除后只剩两项（剩二代一）

（2）怎么用

能排除的优先排除，排除不了再进行代入。

奇偶、倍数、尾数、大小

（3）技巧

年龄题型：年龄差不变，

最大的数为多少/最大可能为XX: 建议从大的开始代入

2. 数字特征法

（1）奇偶特性

  • 加减法

和差同性，a+b与a-b奇偶相同

  • 乘法

一偶则偶，全奇为奇

（2）倍数特性

若A = B * C, 则 A 能被 B 或 C 整除，B 和 C 均是A的约数。

  • 余数型

若答案$= a * x+b$，则答案-b能被a整除。(a、x为整数)

  • 比例型

$A/B = m/n$可以得到：

比例的常见形式：

分数；比例；百分数；倍数

3. 方程法

（1）普通方程

设未知数技巧

  • 设小不设大（减少分数计算）；
  • 设中间量（方便列式）；
  • 问谁设谁（避免陷阱）

前两个快，最后一个准

（2）不定方程

分析奇偶、倍数、尾数等数字特征，尝试代入排除。

奇偶：两数刚好一奇一偶

倍数：a或b与M有公因子

尾数：a或b的尾数是5或0

（3）不定方程组

  • 未知数一定是整数的不定方程（组）

    先消元转化为不定方程，再按不定方程求解。

  • 未知数不一定是整数的不定方程（组）

    一般为时间和钱数

    方法：

    1）赋零法

    对于未知数不一定是整数的不定方程组，可以赋其中一个未知数为0.

    2）配系数
