题目2 最大深度leetcode#104

  • 题目
  • 思路

1)递归：根节点对应的最大深度是左右子树最大深度的最大值加一。

2）广度优先： 最后到达的叶子节点的是最大深度。O(N)

3）深度有限：记录各节点的深度，通过访问到的叶子节点的深度更新max，直到遍历完成。O(N)

题目3 最小深度leetcode#111

  • 题目
  • 思路

1)递归：根节点对应的最大深度是非空左右子树最小深度的最小值加一。

2）广度优先： 第一个叶子节点的深度为最小深度。O(N)

3）深度有限：记录各节点的深度，通过访问到的叶子节点的深度更新min，直到遍历完成。O(N)
