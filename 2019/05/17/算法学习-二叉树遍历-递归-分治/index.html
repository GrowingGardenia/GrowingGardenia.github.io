题目1 计算Pow(x,n) leetcode#50

  • 题目
  • 思路

1）直接调库函数 O(1)

2）暴力，循环乘法 O(n)

3）分治，n个x相乘时，将相乘的算法一分为2，n为偶数时result = x^(n/2) x^(n/2)，n为奇数时result = x^(n/2) x^(n/2) * x

时间复杂度O(log2(n))
